Unbounded

Unlock a world of creativity

Unbounded is an visual html editor for creating websites and components. You can open or copy html files into the editor and modify them without having to write code!

Designers can also use Unbounded similar to how they use tools like Figma, creating designs ready to pass off to developers. ( But also producing code at the same time! )

We designed Unbounded to be close to html. Often website creation tools do some translation from the editor to the actual produced code. This can keep things simple but affords less control and freedom. Unbounded is set up to use the same names for style properties as css to avoid doing any translation. If you are already familiar with html and css then you should feel right at home.

Unbounded is open source and free! We're aiming to make it easier for people to create a web presence for their business, and enable creative people.
